# Profession AI Projects

This repository contains the final projects of the AI Engineering Master. They are classified by topic/course name and project name. A small description of each project is included:

1. Python programming - Python-Powered Contact Management System for ContactEase
   
Brief description: An interactive Python CLI tool that streamlines contact management using Object-Oriented Programming. It features full CRUD functionality, search capabilities, and JSON-based data persistence for a lightweight yet robust organizational experience.

2. C++ programming - Customer Relationship Management (CRM) for InsuraPro Solutions

A robust C++ CRM for insurance firms designed to centralize client data and interaction tracking via Object-Oriented Programming. It manages profiles, sales appointments, and contracts through an interactive CLI with CSV/Text data persistence for optimized operational efficiency.
  
3. Machine Learning fundamentals - Prevision Model for RealEstateAI Solutions

A machine learning project focused on real estate price prediction using Python. It implements and compares Ridge, Lasso, and Elastic Net regularization techniques to minimize overfitting, featuring a full pipeline from data preprocessing to performance evaluation via MSE and residual analysis.

4. Machine Learning models and algorithms - TropicTaste Classification

An automated fruit classification system using Python and the K-Nearest Neighbors (KNN) algorithm. It features a complete pipeline from exploratory data analysis (EDA) to feature scaling and cross-validation, optimizing supply chain efficiency through accurate categorical prediction of exotic fruit types.

5. MLOps and Machine Learning in deployment - for this project, see this [repository](https://github.com/alessiofontanarossa/MachineInnovators_monitoring).
6. Deep Learning and artificial neural networks - Animal & Vehicle Recognition for Autonomous Driving

A deep learning project utilizing Convolutional Neural Networks (CNN) and the CIFAR dataset to distinguish between animals and vehicles. It focuses on real-time safety for autonomous driving, featuring a detailed analysis of model accuracy, precision, and error patterns to optimize urban wildlife monitoring.

7. Applied deep learning with PyTorch - Food Classification & Optimization for GourmetAI

An advanced computer vision project focused on high-accuracy food recognition using Transfer Learning and Data Augmentation. It implements a sophisticated pipeline involving fine-tuning pre-trained neural networks and hyperparameter optimization to achieve robust generalization and production-grade performance.

8. Computer Vision - Flower Recognition for GreenTech

A precision agriculture project using PyTorch and the timm library to automate floral identification. It employs Transfer Learning and advanced data augmentation to distinguish between species like Daisies and Dandelions, optimizing crop health monitoring through high-performance Computer Vision and F1-score maximization.

9. Reinforcement Learning - Healthcare Cyber-Defense via RL for DeepGuard

An advanced cybersecurity project using Reinforcement Learning to protect sensitive medical data. It utilizes the gym-idsgame environment to train agents with SARSA and Double Deep Q-Networks (DDQN), simulating complex attack-defense scenarios to identify network vulnerabilities and automate proactive threat mitigation.

10. Generative AI - Synthetic Augmentation for Critical Infrastructure for CyberEye

A cutting-edge security project enhancing power plant surveillance through Generative AI. It implements a sophisticated pipeline using Image Captioning and Text-to-Image models to synthesize diverse training data, significantly improving model robustness and detection accuracy for anomalies in critical infrastructure monitoring.

11. Explainable AI - Explainable AI (XAI) for Compliance at Banca Virtuosa

A financial sector project focused on the transparency and interpretability of deep learning models. By implementing Grad-CAM, Occlusion Map and Integrated Gradients, it generates saliency maps to visualize decision-making processes, ensuring regulatory compliance and identifying systematic errors in financial data classification.
